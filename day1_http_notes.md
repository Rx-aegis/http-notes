\# Day 1 HTTP Notes



\## What is HTTP



HTTP is a communication protocol between a browser and a server. When a client visits a website, it sends an HTTP request, and the server sends back an HTTP response.



\## Difference between GET and POST



GET and POST are both request methods. A GET request is used to retrieve data from a website, and it usually does not change the website's content or data. A POST request is used to submit data or create something on the website, so it may change the website's content or data.



\## What are 200 / 404 / 500



200 means the request was successful, 404 means the page was not found, and 500 means there is a server error. These are HTTP status codes that tell us what happened to the request.
